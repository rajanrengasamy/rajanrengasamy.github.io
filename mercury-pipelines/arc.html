Pipeline Overview

ARC (Autonomous R&D Cycle) runs nightly at 1 AM AEST. It researches trending topics from two tracks (traditional sources + Reddit pain points), scores and selects the best opportunities, generates detailed PRDs, builds working code, and pushes to GitHub — all while you sleep.

Metric Value
Kickoff Time01:00 AEST nightly
Build Cap6 projects/night (3 traditional + 3 Reddit)
GitHub Push Rate~5/6 repos successfully pushed
Duration~4-20 seconds (inline fallback) to 2+ hours (full build)
Cost$0 (Claude Max + GPT Pro subscriptions)

Traditional Track

Researches AI/agent tools, frameworks, and trending tech from curated sources.

  • AI agent frameworks and tools
  • Coding agent releases (Claude Code, Codex, Cursor)
  • Memory/architecture patterns
  • Open-source AI projects

Reddit Solve-This Track

Finds real user pain points from Reddit and builds solutions.

  • r/LocalLLaMA feature requests
  • r/ClaudeAI workflow pain points
  • "I wish someone would build..." posts
  • Utility tool opportunities
